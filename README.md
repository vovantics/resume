# resume

Stevo's resume generator.

## Installation

```
$ npm i
```

## Usage

### Export as HTML

```
$ npm run export:html
```

### Export as PDF

```
$ npm run export:pdf
```

### Install New Theme

1. Browse themes here: http://jsonresume.org/themes/
1. Pick and install theme: https://www.npmjs.com/search?q=jsonresume-theme
